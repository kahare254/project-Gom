# Backend Setup Instructions

1. Create a folder named `static` inside the `backend` directory.
2. Place the following files in `backend/static/`:
   - `placeholder_1.jpg` (Christian template image)
   - `placeholder_2.jpg` (Muslim template image)
   - `qr-code.png` (QR code image)
3. Install dependencies:
   pip install -r requirements.txt
4. Run the backend:
   python app.py

The API will be available at http://localhost:5000/api/christian-template and /api/muslim-template
